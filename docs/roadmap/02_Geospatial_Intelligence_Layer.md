# 02: Geospatial Intelligence Layer

## Overview
Elevating the AlaskaIntel map from a static visualization to a real-time, multi-layered decision engine.

## Vision
Every signal in AlaskaIntel should be clickable on a coordinate-accurate map with historical context.

## Planned Features
- **Aviation Overlay**: Real-time ADS-B and FAA Sectional integration for situational flight awareness.
- **Dynamic Heatmaps**: High-intensity activity clusters (Emergencies, Events, Economic activity).
- **Historical Playback**: Swipe back in time to see how a wildfire or event unfolded on the map.
- **Custom KML/GeoJSON Ingestion**: Allowing users to drop their own data layers into the Alaska OS.
